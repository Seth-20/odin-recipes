# odin-recipes
Restaurant recipe display website:

This project focuses on use of HTML to organize elements to intuitively display restaurant cooking recipe. Through the project,
I will be demonstrating my understanding of use of HTML elements for creation of text objects, formatted into headings, paragraphs
as well as link embedments, text formatting with special emphasis and bolding. Through this project, I will also demonstrate my 
understanding of acceptable standards in creating formatting HTML documents.
